# Wastewater Treatment Using Multiple CSTRs in Series

A Chemical Reaction Engineering project exploring the optimal design of multiple Continuous Stirred Tank Reactors (CSTRs) for biological wastewater treatment.

## Overview

This project investigates wastewater treatment using multiple CSTRs arranged in series under both isothermal and adiabatic operating conditions. The study incorporates Monod kinetics, Activated Sludge Models (ASM), cost optimization, and environmental considerations to determine the most efficient reactor configuration.

## Objectives

- Model wastewater treatment using multiple CSTRs
- Compare isothermal and adiabatic reactors
- Apply Monod kinetics and ASM
- Optimize conversion vs operating cost
- Determine the optimum number of reactors

## Key Features

- CSTR Design Equations
- Monod Kinetics
- Activated Sludge Model
- Cost Optimization
- Damkohler Number Analysis
- Environmental Regulations
- Aerobic vs Anaerobic Systems

## Results

- Optimal number of reactors: **3**
- Conversion achieved: **94.73%**
- Annual operating cost: **₹65,000/year**
- Demonstrated diminishing returns beyond three reactors.
